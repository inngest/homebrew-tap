# homebrew-tap

## Cask install (recommended)

Trust the cask formula in this tap (if on a new enough version of homebrew):

```
brew trust --cask inngest/tap/inngest
```

Then install the prebuilt Inngest CLI from cask:

```
brew install inngest/tap/inngest
```

or

```
brew tap inngest/tap
brew install inngest
```

## Source build (potentially out of date)
To build from source instead:

```
brew install inngest/tap/inngest-source
```
